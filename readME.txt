ChocoDig!  The premiere addon to help you get your Wing Skill to 100!

This addon will walk around a zone randomly and dig at set time and distance intervals.  It features a GUI to track your progress
and will automatically stop upon zoning, hitting 100 digs, or running out of greens.  It features functionality to randomly change as it digs
and will change direction when it detects that it's stuck.

The following commands are used to operate this addon.  You can either type //chocodig or //cdig for short:

Start            : Loads GUI and begins digging
Stop             : Pauses addon; resume using the Start command.
Pause            : Pauses addon; resume using the Start command.
Reset            : Resets the Dig Counter.
Status           : Shows current status.
Turnreset        : Reset automatic turn cycle.
Distance <number>: Set required movement distance between digs.  Saves between sessions.
Timer  <number>  : Set time between digs in seconds.  Saves between sessions.
Help             : Displays commands
